# 🧾 菜單拍照計算機 — 整合規劃

## 目標
將「自訂計算機」＋「Gemini Canvas 拍照轉 JSON」整合為一個全新獨立工具，店家用手機拍菜單 → AI 自動辨識 → 直接開始點餐計算。原本兩個專案完全不動。

## 使用者流程
```
首頁
 ├── [📸 拍照辨識菜單]
 │    1. 拍照 or 上傳菜單圖片
 │    2. Gemini AI 辨識 → 輸出 JSON
 │    3. 預覽辨識結果（品項、價格、單位）
 │    4. 手動修正（增/刪/改）
 │    5. 按「開始點餐」→ 自動載入計算機
 │
 └── [🧮 直接開始點餐]
       跳過拍照，直接進計算機
```

## 技術架構
單頁應用 (index.html)，兩種模式切換：
- **模式 A：拍照辨識** — 沿用 Canvas 的拍照 + Gemini API 呼叫寫法
- **模式 B：點餐計算機** — 移植自訂計算機核心邏輯
